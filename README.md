# MOKUKU-INS

Inertial Navigation System (INS) algorithms for orientation and pose estimation.

## Modules

| Module | Description |
|--------|-------------|
| [3dof](3dof/README.md) | 3DoF orientation estimation (ESKF) using IMU + scalar velocity. Focus on pitch. |
| [6dof](6dof/README.md) | 6DoF pose estimation (ESIKF) using IMU + GNSS. Algorithm reference. |
